# Video

영상 처리와 Optical Flow 기반 움직임 분석을 실험한 프로젝트입니다.

## Project Overview
이 폴더는 영상 내 움직임을 추적하고 분석하는 기초 기법을 직접 적용해보며,  
OpenCV 기반 영상 처리 흐름과 Optical Flow 개념을 코드로 확인하기 위해 정리한 작업물입니다.

예제 코드 실행에 그치지 않고,  
서로 다른 영상 데이터를 사용해 움직임이 어떻게 포착되는지 확인하고  
분석 코드와 시각화 결과를 함께 살펴보는 데 목적을 두었습니다.

## Goal
- Optical Flow 개념 이해
- 영상 내 움직임 추적 실험
- OpenCV 기반 처리 흐름 학습
- 예제 코드 분석 및 응용

## Directory Structure
- `Optical_Flow/`
  - Optical Flow 관련 예제 및 실험 코드
- `analysis/`
  - 분석용 코드와 실험 영상
- 발표 자료
  - 관련 개념 정리 문서

## Main Work
- Optical Flow 예제 코드 실행 및 분석
- OpenCV 기반 영상 처리 실험
- 입력 영상별 움직임 비교
- 결과 확인 및 기초 분석
- W&B를 활용한 실험 기록 및 분석 흐름 확인

## Tech Stack
- Python
- OpenCV
- Optical Flow
- Video Analysis
- Weights & Biases (W&B)
- MLOps

## What I Learned
이 프로젝트를 통해 영상은 이미지의 단순한 연속이 아니라,  
시간축 정보를 함께 고려해야 분석이 가능하다는 점을 체감할 수 있었습니다.

또한 같은 알고리즘이라도 입력 영상의 특성에 따라  
움직임 추적 결과가 달라질 수 있다는 점을 직접 확인할 수 있었습니다.

추가로 W&B를 활용해 실험 결과를 기록하고 비교해보면서,  
단순 실행을 넘어 실험 관리와 추적의 중요성도 함께 경험할 수 있었습니다.
