# Car_Repair

차량 파손 이미지를 분석하고, 손상 부위 식별과 수리비 예측까지 연결해본 프로젝트입니다.

## Project Overview
이 프로젝트는 차량 손상 데이터를 다루는 전체 흐름을 경험하기 위해 진행했습니다.  
단순히 모델 하나를 학습시키는 데서 끝나지 않고, 데이터 전처리, 마스킹 생성, 이미지 기반 손상 분석, 수리비 예측 실험, 그리고 Django 기반 웹 화면 연결까지 하나의 파이프라인으로 정리했습니다.

## Goal
- 차량 파손 이미지를 학습 가능한 형태로 전처리
- 손상 유형 및 부위를 식별할 수 있는 모델 실험
- 분석 결과를 수리비 예측과 연결
- 결과를 웹 화면에서 확인할 수 있는 형태로 구성

## Directory Structure
- `pre-work/`
  - 데이터 라벨 정리
  - polygon 기반 마스킹 생성
  - 손상/부위 기준 데이터 분리 및 전처리
- `model/`
  - 이미지 기반 모델 실험
  - U-Net 관련 학습/실험 노트북
- `model_fit/`
  - 수리비 예측 및 추가 모델 실험
  - 학습 데이터셋 및 저장된 모델 포함
- `django/`
  - Django 기반 웹 프로젝트
  - 업로드, 결과 확인, 화면 구성 관련 코드 포함

## Main Work
- 손상 이미지 데이터 전처리 및 정리
- segmentation 학습을 위한 마스크 생성 파이프라인 구성
- 손상 유형 분리 및 예측용 데이터셋 생성
- U-Net 기반 이미지 처리 실험
- 수리비 예측 모델 실험
- Django 기반 웹 인터페이스 구성

## Tech Stack
- Python
- Django
- U-Net
- XGBoost
- Computer Vision

## What I Learned
이 프로젝트를 통해 이미지 데이터 전처리와 모델링, 그리고 웹 서비스 연결까지 하나의 흐름 안에서 다뤄보는 경험을 할 수 있었습니다.  
특히 모델 성능 자체보다도, 실제 데이터를 어떻게 정리하고 어떤 형태로 문제를 정의할지 결정하는 과정이 중요하다는 점을 많이 배웠습니다.
