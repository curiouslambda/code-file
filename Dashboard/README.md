# Dashboard

Django 기반 대시보드 구조를 활용해 웹 애플리케이션의 화면 구성과 관리 기능을 다뤄본 프로젝트 폴더입니다.

## Project Overview
이 폴더는 대시보드형 웹 서비스 구조를 익히고,  
인증, 페이지 구성, 관리 화면, 정적 리소스 연결 등 웹 애플리케이션의 기본적인 흐름을 직접 다뤄보기 위해 정리한 작업물입니다.

완전히 새로운 서비스 로직을 처음부터 설계한 프로젝트라기보다는,  
기존 대시보드 구조를 바탕으로 실제 웹 애플리케이션이 어떤 식으로 구성되는지 이해하고 적용하는 데 더 초점을 두고 있습니다.

## Goal
- Django 기반 웹 프로젝트 구조 이해
- 대시보드형 UI 흐름 구성
- 인증 및 기본 페이지 동작 확인
- 관리형 웹 인터페이스 구현 경험 쌓기

## Directory Structure
- `django-black-dashboard_FIN/`
  - Django 기반 대시보드 프로젝트
  - 앱 구조, 인증, 템플릿, 정적 파일 구성 포함
- 기획안 및 문서 파일
  - 프로젝트 방향 또는 관련 자료 정리

## Main Work
- Django 프로젝트 구조 파악 및 적용
- 인증 및 페이지 흐름 구성
- 템플릿 기반 화면 구성
- 대시보드 스타일 UI 적용
- 관리형 웹 서비스 형태 실험

## Tech Stack
- Python
- Django
- HTML / CSS
- Dashboard Template

## What I Learned
이 프로젝트를 통해 웹 애플리케이션은 단순히 기능 구현만으로 끝나는 것이 아니라,  
프로젝트 구조, 템플릿 관리, 정적 파일 구성, 인증 흐름 같은 요소들이 함께 맞물려야 한다는 점을 익힐 수 있었습니다.

특히 대시보드형 인터페이스는 화면 구성이 복잡해질수록  
기능보다 구조를 먼저 이해하는 것이 중요하다는 점을 배울 수 있었습니다.
