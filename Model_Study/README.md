# Model_Study

주요 딥러닝 모델 구조를 직접 구현하거나 따라가며 학습한 스터디 폴더입니다.

## Project Overview
이 폴더는 GAN, Transformer, U-Net 등 대표적인 딥러닝 아키텍처를  
코드 수준에서 이해하고 실험하기 위해 정리한 작업물입니다.
  
모델별 구조와 학습 흐름을 직접 구현하거나 분석하면서  
핵심 개념을 이해하는 데 초점을 둔 학습 폴더입니다.

## Goal
- 주요 딥러닝 아키텍처 구조 이해
- 모델별 핵심 아이디어를 코드로 확인
- 구현 경험을 통해 개념을 정리
- 서로 다른 모델 구조의 특징 비교

## Directory Structure
- `gan/`
  - GAN 구조 관련 코드
- `transformer/`
  - Transformer 학습 자료 및 모델 구조 실습 코드 파일
- `unet/`
  - U-Net 기반 학습 코드

## Main Work
- 모델 구조별 구현 연습
- 코드 기반 개념 이해
- 학습 흐름과 입력/출력 구조 확인
- 아키텍처별 특징 비교

## Tech Stack
- Python
- PyTorch
- GAN
- Transformer
- U-Net

## What I Learned
이 폴더를 통해 모델 이름이나 개념만 아는 수준이 아니라,  
각 아키텍처가 실제 코드에서 어떤 형태로 구현되는지 이해할 수 있었습니다.

특히 모델별로 입력 처리 방식, 네트워크 구조, 학습 목적이 어떻게 다른지 비교하며  
딥러닝 모델을 더 구조적으로 바라보는 데 도움이 되었습니다.
