# Deep_Numpy

딥러닝 프레임워크 없이 수치 계산만으로 모델을 구현하고,  
프레임워크 기반 방식과 비교하며 학습한 기록을 정리한 폴더입니다.

## Project Overview
이 폴더는 딥러닝 모델을 단순히 사용하는 수준을 넘어,  
학습이 실제로 어떤 계산으로 이루어지는지 직접 구현하며 이해하기 위해 정리한 작업물입니다.

California Housing 회귀 문제를 중심으로,  
프레임워크 없이 직접 계산을 구성한 방식과 PyTorch 기반 구현을 비교하면서  
모델 학습의 내부 동작을 더 깊이 이해하는 데 목적을 두었습니다.

## Goal
- 딥러닝 프레임워크 없이 모델 학습 과정 이해
- 순전파와 역전파 계산 흐름 직접 확인
- 회귀 문제를 직접 구현 방식으로 해결
- PyTorch 방식과 비교하며 차이 이해

## Files
- 프레임워크 없이 구현한 회귀 실험 파일
- PyTorch 기반 회귀 실험 파일
- 실험 내용을 정리한 PDF 문서

## Main Work
- 회귀 문제를 직접 계산 기반으로 구현
- 학습 과정 수식과 코드 흐름 확인
- 프레임워크 기반 구현과 직접 구현 비교
- 결과 및 구현 방식 정리

## Tech Stack
- Python
- Numpy
- PyTorch

## What I Learned
이 작업을 통해 모델 성능보다도,  
학습이 실제로 어떤 계산 절차로 이루어지는지 이해하는 것이 얼마나 중요한지 체감할 수 있었습니다.

프레임워크를 사용할 때는 보이지 않던 부분들을 직접 구현해보면서,  
가중치 갱신, 오차 계산, 학습 흐름을 훨씬 구체적으로 이해할 수 있었습니다.
