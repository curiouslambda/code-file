# Spaceship

EDA와 전처리 과정을 중심으로 진행한 데이터 분석 프로젝트입니다.

## Project Overview
이 폴더는 데이터셋을 이해하고,  
결측치 처리와 변수별 특성 분석을 통해 모델링 이전 단계의 데이터 품질을 높이는 데 집중한 작업을 정리하고 있습니다.

단순히 예측 모델을 빠르게 적용하기보다,  
데이터를 먼저 충분히 해석하고 전처리 전략을 비교해보는 과정에 더 큰 비중을 두었습니다.

## Goal
- 데이터셋 구조 파악
- 주요 변수의 분포 및 관계 분석
- 결측치 처리 전략 비교
- 모델링 이전 단계의 데이터 정제

## Directory Structure
- `CSV/`
  - 원본 및 전처리된 데이터 파일
- `DataOverview/`
  - 데이터 전체 구조 요약
- `EDA/`
  - 변수별 탐색적 데이터 분석
- `Preprocessing/`
  - 결측치 처리 및 전처리 실험

## Main Work
- 데이터 전체 구조 파악
- 주요 변수별 시각화 및 관계 분석
- 결측치 처리 방식 실험
- 전처리 결과 비교 및 정리
- 모델링 전 데이터 품질 개선

## Tech Stack
- Python
- Pandas
- Data Analysis
- EDA

## What I Learned
이 작업을 통해 모델링 이전에 데이터를 얼마나 깊게 이해하느냐가  
후속 결과에 큰 영향을 준다는 점을 확인할 수 있었습니다.

특히 결측치 처리는 단순 보정이 아니라,  
변수의 의미와 데이터 분포를 같이 고려해야 한다는 점을 많이 배웠습니다.
